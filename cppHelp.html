第一部分 C++

第一章 注释

很多人很奇怪，为什么要先学注释？答案只有一个：简单。
在C++中，注释的语法是两个英文的斜杠//。注意要点：

  1. 斜杠要是英文的（这个不仅仅针对注释，在C++中，所有符号都要是英文的）
  2. 斜杠有正斜杠/和反斜杠\，别打错了

注释的功能是让编译器忽略双斜杠后面的内容，也就是说，这一行在双斜杠后面的任何文本，无论是英文还是中文，都会被忽略，但是双斜杠前面的文本以及不在这一行的文本都不会被忽略。
下面看一个示例。在示例中，callFunction();是一个C++语句，这里大家不需要知道语句的意思，只用知道他是一个语句就行了。
前面说的再强调一遍，从这里开始，一直到教程说“代码可以运行”为止，教程中的示例都不能运行。

//这是一个注释，从这里开始一直到此行的结尾，都会被忽略

另外一个示例标明了注释的作用范围

callFunction();//这是一个注释

在这个示例中，callFunction这个语句仍然会被执行，因为//注释只忽略//后面的内容，不忽略前面的内容。

再来看一个示例

//callFunction();

这个时候，callFunction();语句不会运行，因为他写在了//的后面，//后面的一切内容都会被忽略。

再来看一个示例

callFunction();
//这是一个注释
callFunction();

在这个示例中，两个callFunction都可以被运行，因为注释只影响一行，不影响上下的行。
在DevC++软件中，被注释忽略的地方会用绿色标出，没有被忽略的地方是其他颜色。（如下图）

另外，还有一种“多行注释”，需要/*和*/来实现，被忽略的范围在/*和*/之间。这种注释不细讲，见到认识就行，不推荐使用。

/*这是一个多行注释
这几行都会被忽略
啦啦啦啦啦啦
一直到这里结束*/
callFunction();//这行语句可以正常运行

第二章 变量初步

2.1 数据类型初步

本章初步阐述了数据类型。详细的数据类型在后面的章节会另述。

2.1.1 数据类型

在生活中，有许多数据，例如车牌号，电话号码，一串文本，一个数字……
因此，在C++中定义了“数据类型”，描述了一个数据的类型以及储存方式

数据类型名称 中文名称 详细介绍
int 整型 储存一个整数，例如5或者3，不能存储小数，默认范围是-2147483648~2147483647
float 浮点型 储存一个小数，精确到小数点后7~8位，例如3.14或者5.0
double 双精度浮点数 储存一个小数，精确到小数点后15~16位，可以看做是float的双精度版本（即精度x2），例如3.14159265358979或者3.0
char 字符 存储一个ASCII字符（包含英文大小写字母，数字，英文符号以及一些特殊的控制字符，后面会详细展开）。注意：char类型只能储存一个ASCII字符，不能存储中文以及多个字符
bool 布尔 存储 真（true）或者 假(false)，也就是说这个类型只能存储两种情况。

2.1.2 数据类型修饰

根据上面的表格，我们发现int类型默认储存范围是-2147483648到2147483647，也就是说只能存储-2147483648到2147483647之间的数。有的时候我们需要存储更大的数字，有的时候我们不需要这么大的容量，希望减少储存范围来节约内存，有的时候我们不需要储存负数，只希望储存正数。这个时候我们就需要数据类型修饰符来修饰类型。

修饰符 名称 详细说明
singed 有符号 标记某个类型的储存范围包含负数和正数。这个标识符可以省略，因为int，float和double默认就是有符号的
unsigned 无符号 标记某个类型不储存负数，只储存正数。例如，int类型默认是-2147483648到2147483647，总共能储存的范围有4294967296个数字，unsigned可以让int的储存范围变成0~4294967295，也就是说通过舍弃储存负数来让他拥有储存更大正数的能力
long 长 以使用更多内存空间为代价，让某个数据类型能够存储更大的值，例如int的储存范围是-2147483648~2147483647,加上long后储存范围从-9223372036854775808到9223372036854775807，更大了。
short 短 通过变小某个数据类型的储存范围，来节约内存。例如short修饰int就可以让储存范围变成-32768~32767

那么如何进行修饰呢？只需要在类型名前面打上修饰符就行了。
例如，要让unsigned修饰int，我们只需要输入unsigned int就行了（注意unsigned和int中间有空格）
在C++中，类型名和修饰都严格区分大小写，上述关键字必需小写，否则编译器无法识别

2.1.3 数据的写法

我们知道了数据类型，也要知道数据的写法。

  • int类型的数据如何在程序中写呢？在程序中，int类型的数据直接是一个整数，例如5或者3
  • double类型的数据如何在程序中写呢？在程序中，double类型的数据直接可以表示为一个小数，例如3.14
  • float类型的数据和double类型差不多，但是float类型的数据在写的时候需要在后面加一个f，例如3.14f或者1.23f或者1.0f（注意，必需有.0，直接写1f是错的）
  • char类型的数据需要用单引号（注意，只能是英文的单引号，英文的双引号或者中文符号都是不行的）例如'a'，'6'等。另外，char类型只能表示一个字符，所以不能表示两个字符，也不能表示中文字符。所以'aaaaa'或者'啊'都是不行的
  • bool类型在程序中只有两个值，一个是true，一个是false

在devC++中，int,double和float类型是草绿色，而char类型是橘色。

2.2 标识符

在代码中，有一些名称，例如函数名、变量名（详见后文）是可以自定义的，那么这些自定义的名称有规范吗？有的。
下面是标识符规范：

  1. 标识符只能包含大写英文字母A-Z，小写英文字母a-z，数字0-9，以及英文的下划线_
  2. 标识符不能以数字开头

由上可知，标识符禁止包含空格、中文以及其他符号。
因此，标识符命名尽量用英文。例如我需要表示字符个数，那么英文就是number of characters，因为不允许包含空格，所以就删掉空格变成numberofcharacters。这是一个符合规定的标识符，但是可读性太差，所有单词连到一起，不方便辨认。因此，我们可以使用下划线或者大写字母改变。
方法1：在每个单词之间添加下划线_，因为标识符允许出现下划线，所以上面的那个标识符就可以命名为number_of_characters，是不是好辨认了？
方法2：每个单词首字母大写，上面的标识符可改为NumberOfCharacters，是不是也很好辨认？
但是请注意一点：上面两种方法都是建立在符合标识符命名规范的基础上的，符合规范是前提，否则编译器无法识别！
在DevC++中，符合规定的标识符用天蓝色标识

当我们声明变量或者函数的时候（见后文），实际上是创建了一个标识符。重点来了：除了后面讲的作用域特殊情况以外，标识符不能重名。也就是说，如果你声明了一个名字叫做abc的变量（声明变量的语法见后文），那么你就不能再定义一个名字同样叫做abc的函数（见后文）

2.3 变量初步（概念）

在程序中，我们需要储存一些数据，例如电话号码，一个数字，一个字符，一个小数……这个时候，我们就可以使用变量。变量可以储存这些数据，并进行修改，读取，赋值和删除。
在声明变量之前，我们需要先思考，这个变量需要使用哪个数据类型。例如，我需要储存字符个数，那么我就应该用int类型。如果我要存储小明的身高（可能不是整数），就需要使用float或者double类型。如果我需要存储“你最喜欢的英文字母”，就需要使用char类型。如果要存储你是否喜欢苹果这个信息，就需要使用bool类型。
然后，你需要给变量想出一个名字。在以后使用、修改、读取这个变量的时候，都需要使用这个名字。值的一提的是，这个名字属于标识符，需要遵循标识符命名规范

一切都想好了，就可以开始编写程序。
在使用变量之前，你需要先告诉编译器你的变量叫什么名字，他需要存储什么数据类型。这个告诉的过程就叫做变量声明。变量声明的语法如下：

数据类型 变量名;

要注意几个点：

  1. 数据类型是前面讲过的int，double等等。
  2. 数据类型和变量名之间要有一个或者多个空格
  3. 变量名需要满足标识符命名规范
  4. 最后也是最重要的，每个语句后面必需有英文的分号！！！其实不只是变量声明，任何C++语句都必须在结尾处打一个英文的分号

例如你想要声明一个int类型，名字叫做“number_of_characters”的变量，就需要这样写：

int number_of_characters;

如果你需要声明一个bool类型，名字叫is_key_A_pressed的变量，就需要这样写：

bool is_key_A_pressed;

例如你想要声明一个unsigned int类型，名字叫做“age”的变量，就需要这样写：

unsigned int age;

声明完成之后，就可以使用了，但是对于变量的使用，这里暂且不详细论述。等讲完函数之后再进行论述。

第三章 函数

3.1 函数概念

我们先来假设一个场景：你正在给一个机器人编程序，让他走路。你写了一段代码如下
身体：面朝 充电桩方向
左腿：往前5cm
左腿：向下施加20N的力
右腿：向上5cm
右腿：向前15cm
右腿：向下施加20N的力
……


你就这样给他编写了一个走到充电桩充电的程序。但是如果你下次要在后面增加一个去厕所拉屎的程序，又得复制这一整段代码，然后修改面朝的方向，继续运行。如果你下一次又要增加一个去别的厕所的程序，又得重新复制粘贴。最尴尬的来了，如果你想要把“施加20N的力”改成施加30N的力，你就得把刚刚复制粘贴的所有20N改成30N，十分麻烦。
于是，我们想，能不能把这些重复的，需要复制粘贴的语句将其“打包”起来，每次需要调用重复语句的时候，就可以只用写一句话，编译器自动会调用你“打包”出来的所有语句？可以，这就是函数。下面再来看一个示例：

定义一个函数：走一步路（请输入：int类型的 距离 和 int类型的 方向，0°到360°）{
    身体：面向 （刚刚输入的 int类型的方向）
    左腿：向前 （刚刚输入的int类型的距离） cm
    右腿：......
    ......
}

然后，在后续的代码中就可以：

调用函数：走一步路（距离：5，方向：135°）;

此时，就相当于在调用函数的地方粘贴了函数内部的代码。这就是函数。

到这里请先停一下，先确保你已经完全理解了函数的概念和用途，再往下看。

3.2 函数的定义

在定义函数前，我们需要先明确几个事项：

  • 函数的名称叫什么？（函数的名称遵循标识符命名规范）
  • 函数的返回值是什么数据类型？有些函数像我们刚刚编写的机器人走路函数一样，是进行一些操作，然而有一些函数，例如“获取当前电量”不仅需要执行一些操作，还需要告诉函数的调用者一个具体的数据。这个数据就叫做返回值，返回值的数据类型必需确定。有些函数例如刚刚的机器人走路函数不需要返回值的，那么返回值的数据类型就是void来告诉编译器这个函数没有返回值
  • 函数需要接收什么样的参数？例如刚刚的机器人函数，输入int类型的距离和int类型的方向就是参数。参数写在函数定义的尾部的一个括号里面，这个括号里面的所有东西就叫做传参列表。

然后我们来学习函数定义的语法：

函数返回值数据类型 函数名(参数1类型 参数1名称,参数2类型 参数2名称,......){
    函数体
}

函数返回值数据类型可以是void（代表没有返回值），int或者是上面“数据类型”那一章提到的其他任何数据类型
函数内的代码和语句必须写在英文的大括号{和}之间，否则这个语句就不在函数内

函数返回值数据类型 函数名(参数1类型 参数1名称,参数2类型 参数2名称,......){
    语句写在这里是函数内，属于这个函数
}
语句写在这里就不属于上面的函数

在这里还要强调一下：在C++中，所有大括号，中括号，小括号都必须配对，单独的括号是错的！
例如:

{{{{ }}}}//这些括号是对的
{{{ ((( [[[ ([ ]) ]]] ))) }}}//这个也是对的
{{}//这个是错的！左边多出来了一个{
{(})//这个是错的！反括号的顺序错了！应该改为{()}

函数名后面的括号里面就是传参列表，分别写上参数的类型，加上一个空格后写参数的名称，多个参数用英文的逗号分割。如果没有参数，就把括号留空，例如：

函数返回值数据类型 函数名(){
    函数体
}

不难看出，其实传参列表就是变量声明的语法，只不过变量声明后面不用分号，而是用逗号。声明的参数其实就是变量，可以在函数内当做变量来使用。
下面我们尝试把刚刚的机器人走路函数翻译成C++。
首先我们先来确定如下内容：

  • 返回值类型：无返回值（void）
  • 函数名；robotMove
  • 参数：
    • 距离：distance（int类型）
    • 方向：direction（int类型）
      所以我们就可以写出如下代码：
void robotMove(int distance,int direction){
    //在这里写上让机器人移动的代码
}

如果我们要写一个机器人跳跃的函数，一样分析：

  • 返回值类型：无返回值（void）
  • 函数名；robotJump
  • 参数：无参数
    可写出：
void robotJump(){
    //在这里写上让机器人跳一下的代码
}

如果我们要写一个返回值为5的函数，那么就可以：

int return5function(){
    return 5;
}

这里用到了return 语句，我们稍后会讲解

3.3.1 函数的调用

学会了定义函数，接下来可以调用它了。
如果我们还是定义了机器人移动的函数

void robotMove(int distance,int direction){

如何调用它呢？

首先，先写我们需要调用的函数的名字robotMove，然后再加上一个括号。括号里面写上函数的参数，和函数声明中的参数一一对应。在上面这个示例中，distance就是5，direction就是270。我们需要更直观的体现函数的作用：

void robotMove(int distance,int direction){

3.5 函数声明

我们可以看一个函数：

void myFunction(){

}

int main(){
    myFunction();
    return 0;
}

这个代码是正常的。但是现在我有一个要求：把myFunction的定义移动到main函数的后面

int main(){
    myFunction();
    return 0;
}

void myFunction(){

}

这个时候编译器会报错:[Error] 'myFunction' was not declared in this scope

这是因为编译器在编译main函数的时候，还没有看到myFunction的定义，所以他根本不知道myFunction是什么鬼东西，就报错。
于是我们可以进行函数声明，简单来说就是告诉编译器“myFunction是个函数，有这个东西，你先别急着报错，等你看完整个代码之后你就能找到定义了”
和函数定义一样，函数声明也不能写在函数的里面，必需写在函数外面。
函数声明的格式是：

函数返回值类型 函数名(传参列表);

其实函数声明就是函数定义把大括号换成分号就行了。
值的注意的是，函数声明的返回值类型，函数名和传参列表必需和函数定义完全一样
因此，上面的代码可以改为：

void myFunction();  //函数声明

int main(){
    myFunction();
    return 0;
}

void myFunction(){

}

之后代码可以正常运行了。

函数是不能重复定义的，例如：

void myFunction();  //函数声明

int main(){
    myFunction();
    return 0;
}

void myFunction(){

}
void myFunction(){//错误！重复定义了myFunction函数！

}

但是函数声明是可以重复的。 
void myFunction();  //函数声明
void myFunction();  //函数声明第二遍
void myFunction();  //函数声明第三遍
void myFunction();  //函数声明第四遍
//还可以继续增加，都没有问题。

int main(){
    myFunction();
    return 0;
}

void myFunction(){

}
